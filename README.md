# Steady Study :blue_book:


## Our Direction
저희 스터디는 꾸준한 목표 성취와 학습을 추구합니다. 
- 목표를 설정하고 브리핑을 하며 스스로를 평가합니다.
- 온라인 스터디를 통해 자율 학습을 독려합니다.

</br>

## Study Rules
- 주중 - 디스코드 온라인 스터디 (화상 + 화면공유) 
- 주말, 공휴일 - 휴일
- 지각 3회 누적 또는 무단 결석 1회 벌금 5만원 (매월 1일 초기화)
- 최소 하루 전에 미리 공지하는 경우 사유로 인정

</br>
